# Comic Reader

Simple comic reader app built with React, Vite, Tailwind CSS, and Supabase. It currently supports browsing comics, viewing details, and reading pages.

## Features
- Browse comics from Supabase
- Comic detail with chapter list
- Reader view with next/previous navigation

## Next Steps

## Tech Stack

- Tailwind CSS
- Supabase (Auth, Database, Storage)
